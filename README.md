# LyricFinder App

> App that finds song lyrics using React and the Musixmatch API.

## Run Locally

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start
```

## Demo

This app is deployed on netlify. Try it [LyricFinder](https://adorable-medovik-467b6b.netlify.app/)
